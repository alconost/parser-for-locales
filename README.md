Установка:
 
npm install parser-for-locales

Для запуска скрипта добавьте в package.json файл проекта скрипт, примерно так:

 "scripts": {
 
    ...,
    
    ...,
    
    "parser": "node ./node_modules/parser-for-locales/index.js ./src/ ./src/locales/ ./src/"
    
  },

- node ./node_modules/parser-for-locales/index.js - путь к скрипту
- ./src/ -  первый параметр(обязательный), путь к папке в которой нужно искать
- ./src/locales/ - второй параметр(обязательный), путь к файлу или папке с json файлами с переводами
- ./src/ - третий параметр(необязательный), путь к папке в которой будет создан json файл (unusedKeys.json) c неиспользуемыми ключами. (если параметр не указан, файл появиться в корне проекта)

Когда все установлено и настроено нужно прописать npm run parser и, если все пути к файлам и папкам указаны верно, запуститься скрипт и по завершению появиться файл unusedKeys.json

ВНИМАНИЕ: Если в заданных папках будут присутствовать файлы без расширения, скрипт не сработает!
